    Create Python 3.5 Environment

    If you have installed Anaconda with Python 3 ensure it is Python 3.5.5 or lower. Otherwise, you will run into issues with PySpark which is still not compatible with Python 3.6 and above. To check which version you’re currently using execute the following in a terminal:

    python --version

    To use Python 3.5.5 instead, we’ll create a conda environment:

    conda create --name py35 python=3.5.5

    Then to use this environment we’ll run:

    source activate py35

    (py35) should now appear at the beginning of your terminal name. We can then use python --version again to ensure that we’re now using Python 3.5.

    Java with Homebrew on MacOS

    Today, the latest version of Java JDK is 10. For PySpark and Zeppelin to work we need Java 8. If it’s not already available on your machine, you’ll need to install it, the easiest way is via Homebrew. To search if it’s available, type the following:

    brew cask search "java*"

    If java8 does not appear, we’ll need to tap the versions as follows:

    brew tap caskroom/versions

    Then repeat ```brew cask search “java*”. This time java8 does appear. To install, run:

    brew cask install java8

    Once it’s successfully installed we should see two java versions now available when we run:

    /usr/libexec/java_home -V

    The default is still set to Java 10. To switch to Java 8 export the path:

    export JAVA_HOME=$(/usr/libexec/java_home -v 1.8)

    Now when we check with java version (which is slightly different to Java 9/10 java --version) we should see it’s Java 8

    java -version
