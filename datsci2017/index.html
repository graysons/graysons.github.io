    Guest Post for DatSci Awards 2017

    The first question when writing this blog post was ‘What should I write a blog post about?’… To help answer this I decided to explore what other people had written about. Which then answered my first question. So I’m going to write a blog post about what other people have written blog posts about…

    Collect The Data

    First, lets collect the data. To do this, I used a python package called Scrapy:

    Scrapy is an application framework for crawling web sites and extracting structured data which can be used for a wide range of useful applications, like data mining, information processing or historical archival.

    Scrapy has really good documentation and the code below is just an adaptation of the sample spider they walk through on their tutorial page.

    import scrapy

class BlogSpider(scrapy.Spider):
    name = "blogs"
    start_urls = [
        # Link to home of all blog posts.
        'https://www.datsciawards.ie/news/',
    ]

    def parse(self, response):
        # Start on main blog page.
        all_posts = response.css("article")
        # Extract links to page of posts themselves.
        for href in all_posts.css("div.fusion-post-wrapper a.fusion-read-more::attr(href)"):
            # Follow links to post page to extract content.
            yield response.follow(href, self.parse_blog)

    def parse_blog(self, response):
        def extract_with_css(query):
            return response.css(query).extract()

        yield {
            'title'  : extract_with_css('article h1.entry-title::text'),
            # Third item in list is date for all pages.
            'date'   : extract_with_css('div.fusion-meta-info-wrapper span::text')[2],
            # * will visit all inner tags of p (<span><strong> etc) and get text.
            'content': extract_with_css('article p *::text')
        }

    Once you follow the guideline for setting up a new project and save the code above as your spider, just run:

    scrapy crawl blogs -o blogs.json

    in your terminal in the directory of your spider project. This will save the title, date, and text content of each post in a json format in the file called ‘blogs.json’.
